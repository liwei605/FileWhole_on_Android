## FileWhole on Android

FileWhole是一个全文检索软件，包含Windows版，安卓版和鸿蒙版

本仓库为FileWhole的安卓Demo版本

具体使用方式：

1.拉下代码，配置环境：环境信息如下：jbr-21,Gradle-8.13,Android API 35+

2.模拟器编译运行

3.打包发布


技术架构：
目前的安卓只支持Sqlite FTS4版本（FileWhole PC版本为FTS5，可以自定义分词器）
基于FTS插件，我们可以实现100多种类型的文件内容检索，包含（文档文件、图片文件、音频文件等）
